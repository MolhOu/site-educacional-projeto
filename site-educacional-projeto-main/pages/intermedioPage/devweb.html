Desenvolvimento Web

Introdução ao HTML
HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação que tem uma natureza simples, composta por elementos que podem ser aplicados a partes de texto para atribuir-lhes significado específico em um documento. Esses elementos permitem estruturar um documento em seções lógicas, definindo sua organização e hierarquia. Por exemplo, é possível criar parágrafos, listas de seleção, tabelas e outros elementos para melhor organizar e formatar o conteúdo. Além disso, o HTML possibilita incorporar conteúdos multimídia, como imagens e vídeos, em uma página da web, tornando-a mais rica e interativa para os usuários. Neste módulo introdutório, serão abordados os conceitos fundamentais do HTML, bem como a sintaxe básica necessária para compreender como essa linguagem funciona. Com esses conhecimentos iniciais, os usuários poderão começar a criar e desenvolver páginas da web, aplicando a estruturação lógica e visualização de conteúdo de acordo com suas necessidades e objetivos. O HTML é a base da construção de páginas da web e é essencial para qualquer pessoa que deseje se envolver com desenvolvimento web e design de sites.

O que é HTML ?
O HTML (HyperText Markup Language) não é uma linguagem de programação, mas sim uma linguagem de marcação. Seu propósito principal é fornecer instruções ao navegador sobre como estruturar e apresentar o conteúdo de uma página da web que está sendo visitada. Os desenvolvedores web utilizam o HTML para criar páginas com diferentes níveis de complexidade, podendo variar de páginas simples a páginas mais elaboradas. Através do HTML, é possível definir a organização do conteúdo, a formatação e a aparência visual da página. O HTML é composto por uma série de elementos (tags) que permitem anexar, envolver ou marcar diferentes partes do conteúdo, tais como texto, imagens, links e elementos multimídia. Cada elemento possui uma função específica, e o navegador interpreta essas marcações para exibir o conteúdo da página de acordo com as instruções fornecidas pelo desenvolvedor.

As partes principais do elemento são:

  1. Tag de abertura: Consiste no nome do elemento ( neste caso: p ), envolvido entre parênteses angulares de abertura e fechamento. Isso indica onde o elemento começa, ou inicia a produzir efeito — neste caso, onde o parágrafo se inicia.

  2. Tag de fechamento: É o mesmo que a tag de abertura, exceto que este inclui uma barra diagonal antes do nome do elemento. Indica onde o elemento termina — neste caso, onde fica o fim do parágrafo. Falhar em incluir o fechamento de uma tag é um erro comum para iniciantes e pode levar a resultados estranhos.

  3. O conteúdo: Este é o conteúdo do elemento, que neste caso é somente texto.

  4. O elemento: A tag de abertura, mais a tag de fechamento, mais o conteúdo, é igual ao elemento.

O que está no cabeçalho? Metadados no HTML
O cabeçalho (ou header) de um documento HTML é uma parte essencial, mas não visível, que contém informações importantes sobre a página e não é mostrada diretamente no navegador quando a página é carregada. Ele é definido entre as tags e no código HTML. As principais informações que podem ser encontradas no cabeçalho incluem:


  • Title: Define o título da página, que é exibido na barra de título do navegador e também pode ser exibido nos resultados de busca.

  • Links para CSS (Cascading Style Sheets): Os links fornecem referências para arquivos CSS externos que definem a aparência e o estilo do conteúdo HTML na página.

  • Links para favicons personalizados: Favicons são pequenos ícones exibidos na aba do navegador e na barra de favoritos, permitindo uma identificação visual rápida da página.

  • Metadados: São informações adicionais sobre o documento, como autor, descrição, palavras-chave e codificação de caracteres, que auxiliam em mecanismos de busca e indexação do conteúdo.

    • O cabeçalho é crucial para otimizar a experiência do usuário e melhorar a indexação da página pelos mecanismos de busca, além de fornecer informações importantes sobre o conteúdo e a formatação do documento HTML. É uma parte fundamental do desenvolvimento web que ajuda a tornar as páginas mais acessíveis, funcionais e amigáveis.

    Primeiros passos com CSS

    Introdução ao CSS
    CSS ( Planilhas de estilo em cascata) é usada para estilizar e arranjar páginas web — por exemplo, para alterar a fonte, cor, tamanho e espaçamento do seu conteúdo, separá-lo em multiplas colunas, ou então adicionar animações e outras implementações decorativas. Esse módulo provê um começo sutil em seu caminho pelo domínio do CSS com o básico de como ele funciona, como é a aparência da sintaxe e como você pode começar a utilizá-lo para estilizar seu HTML.

    O que é CSS?
    CSS (Cascading Style Sheets) permite que você crie páginas web com ótima aparência. Mas como isso funciona por debaixo dos panos? Este artigo explica o que é CSS, com um exemplo simples de sintaxe, e também cobre alguns termos importantes sobre a linguagem.

    Adicionando CSS para o nosso documento
    A primeira coisa que precisamos fazer é falar para o HTML que temos algumas regras CSS que desejamos usar. Há três formas diferentes de aplicar CSS a um documento HTML que você normalmente vai encontrar, contudo, por enquanto, iremos olhar para o mais habitual e útil – vincular o CSS a partir do cabeçalho do seu documento. Crie um arquivo na mesma pasta em que se encontra o seu documento HTML e salve-o como style.css. A extensão .css nos informa que se trata de um arquivo CSS.
