Lógica de Programação I

O que são algoritmos?
Algoritmos são um conjunto de instruções ou passos bem definidos e ordenados que descrevem uma sequência de ações a serem seguidas para resolver um problema ou executar uma tarefa específica. Em termos simples, os algoritmos são como receitas ou mapas que fornecem um roteiro claro para realizar uma determinada ação. Os algoritmos são usados em diversas áreas, incluindo ciência da computação, matemática, engenharia, economia e muitas outras. Eles são especialmente importantes na área da ciência da computação, onde são amplamente utilizados para projetar e implementar programas de computador. Para que um algoritmo seja eficiente, ele deve ser claro, preciso, lógico e determinístico, o que significa que ele deve fornecer um resultado consistente e correto em todas as circunstâncias. Além disso, os algoritmos são frequentemente avaliados com base em sua eficiência em termos de tempo de execução e uso de recursos computacionais, buscando-se encontrar soluções que sejam rápidas e requeram menos recursos para resolver um determinado problema.

Como funciona a lógica de programação?
Os algoritmos são conjuntos de instruções que representam uma receita ou um plano para resolver um problema específico. Eles fornecem a base para a lógica de programação, que é a forma como essas instruções são apresentadas e organizadas. A lógica de programação estabelece as regras e conceitos que sustentam os códigos que o computador deve ler, interpretar e executar. A partir da estruturação dessa sequência coerente de passos, os aprendizados posteriores são desenvolvidos, incluindo o conhecimento de diferentes linguagens de programação. Por exemplo, linguagens de programação como C, JavaScript e Python são alguns exemplos de conhecimentos que podem ser adquiridos após aprender a lógica de programação. Essas linguagens são usadas para implementar os algoritmos e desenvolver programas de computador para diversas finalidades. A lógica de programação é, portanto, a base essencial para a compreensão e o domínio das linguagens de programação e para a criação de soluções computacionais eficientes e funcionais.

Aprender sobre algoritmos
Aprender a dar comandos claros e coesos para o computador é uma das etapas iniciais essenciais para dominar a lógica de programação. Como mencionado anteriormente, um algoritmo representa uma receita ou um plano que fornece orientação sobre quais ações devem ser executadas e em que sequência. Um exemplo clássico disso é a receita de bolo, onde cada passo é seguido rigorosamente, incluindo ingredientes, quantidades, tempo de preparo e temperatura, para obter um resultado satisfatório. Essa analogia ilustra como o computador funciona, pois, da mesma forma, é preciso indicar cada passo a ser executado com clareza e objetividade para que a máquina possa realizar as tarefas adequadamente. Conhecer os algoritmos é o ponto de partida para aprender lógica de programação. Ao criar algoritmos bem estruturados e precisos, os programadores podem guiar o computador a executar as tarefas necessárias para alcançar um resultado específico. Portanto, a lógica de programação envolve a habilidade de conceber algoritmos coerentes, seguindo uma sequência lógica de ações, e comunicar esses passos de forma clara para que o computador compreenda e execute as instruções corretamente. Dominar essa habilidade é fundamental para aprofundar o conhecimento em programação e desenvolver soluções eficientes e funcionais.

Fluxograma

Conceito de fluxograma
Um fluxograma é um diagrama que representa visualmente um processo, sistema ou algoritmo de computador. Esses diagramas são amplamente utilizados em várias áreas para documentar, estudar, planejar, melhorar e comunicar processos complexos de forma clara e fácil de entender. Os fluxogramas utilizam diferentes formas geométricas, como retângulos, ovais e diamantes, para representar diferentes tipos de passos ou etapas do processo. Setas são usadas para definir o fluxo e a sequência das ações. Esses diagramas podem ser desde gráficos simples desenhados à mão até diagramas mais abrangentes criados por computador, detalhando todas as etapas e rotas do processo. Devido à sua versatilidade, os fluxogramas são comuns em diversas áreas de atuação e são utilizados por pessoas técnicas e não técnicas. Eles também podem receber nomes mais especializados, como fluxogramas de processo, mapas de processos, fluxogramas funcionais, mapeamento de processos de negócios e notação de modelagem de processos de negócio (BPMN, em inglês). Além disso, eles estão relacionados a outros diagramas amplamente usados, como diagramas de fluxo de dados (DFDS) e diagramas de atividade da linguagem de modelagem unificada (UML, em inglês). Em resumo, os fluxogramas são uma poderosa ferramenta de visualização e comunicação que auxilia na compreensão e no aprimoramento de processos complexos, sendo uma valiosa aliada em diversos contextos profissionais e técnicos.

Portugol

Introdução
O Portugol é uma pseudo-linguagem algorítmica muito utilizada na descrição de algoritmos, especialmente para iniciantes no aprendizado de lógica de programação. Sua principal característica é o uso de comandos em português, o que facilita o entendimento e habitua os iniciantes com o formalismo de programação.

Embora o Portugol ofereça vantagens significativas no aprendizado, como a familiaridade com a lógica de programação, apresenta a desvantagem de não permitir a execução dos algoritmos no computador. Isso significa que os iniciantes precisam imaginar a execução passo a passo, o que pode ser um desafio para quem está começando.

A lógica para programação envolve aprender a pensar na mesma sequência em que o computador executa as tarefas. Isso requer treinamento para imaginar como as ações serão realizadas, partindo do estudo do problema e chegando à construção do algoritmo como solução. Um exemplo simples seria o problema de expressar o resultado da soma de dois valores.

Uma pessoa iniciante pode pensar em algo como: "Pegar os dois valores, somar e obter o resultado". Com a experiência adquirida, essa pessoa poderá converter automaticamente esse pensamento em instruções específicas sem a necessidade de especificar detalhadamente cada etapa implícita nessa pequena rotina.


Portanto, o uso do Portugol pode ser uma abordagem eficaz para familiarizar-se com a lógica de programação, mas a transição para linguagens de programação reais permitirá a execução dos algoritmos e o desenvolvimento de habilidades mais práticas no campo da programação


Ações Operações
PEGAR Receber os dois valores numérico e armazená-los
SOMAR Executar a instrução da soma e armazenar o resultado
RESULTADO Mostrar o resultado, armazená-lo para uso posterior ou para ser visualizado em outra oportunidade

O domínio da lógica de programação é fundamental para o desenvolvimento de algoritmos eficientes e para a criação de soluções bem estruturadas para os problemas propostos. Quanto mais familiarizado com os conceitos de lógica e com o processo de pensamento sequencial do computador, mais fácil será detalhar as tarefas envolvidas na solução de um problema e implementar um algoritmo funcional. Para um iniciante, construir um algoritmo que permita ao computador executar uma tarefa pode ser desafiador. Inicialmente, pode ser necessário um esforço maior para entender a lógica envolvida e para imaginar as etapas necessárias para resolver o problema proposto. A transição do pensamento em linguagem natural para o formato estruturado de um algoritmo pode ser uma tarefa não tão simples no início. No entanto, com prática e dedicação, o iniciante pode gradualmente desenvolver suas habilidades de lógica de programação e se familiarizar com os conceitos fundamentais da construção de algoritmos. A prática constante, a resolução de diferentes problemas e a análise de algoritmos já criados podem ajudar a aprimorar essas habilidades. É importante lembrar que o aprendizado de programação é um processo contínuo e que a persistência é essencial para superar os desafios iniciais. Com o tempo e a prática, a construção de algoritmos se torna uma habilidade cada vez mais natural e poderosa para resolver problemas e desenvolver aplicações computacionais.

A tradução de um algoritmo escrito em Portugol para um programa de computador numa linguagem de programação é muito fácil e clara, facilitando assim o ensino e aprendizado da linguagem de programação. O Portugol é a forma mais utilizada para escrever algoritmos por ser uma linguagem simples e permitir o detalhamento dos algoritmos. O ensino do Portugol, hoje, é feito de maneira manual (utilizando folhas de papel), o que não estimula os alunos em aprender e exercitar o desenvolvimento de algoritmos. Este trabalho apresenta um ambiente para o desenvolvimento de algoritmo em Portugol com o auxilio do computador.
