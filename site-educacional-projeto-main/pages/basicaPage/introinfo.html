Introdução á Informática

Ciclo de funcionamento de um PC:
Ao ligar o computador, ocorre a busca na memória ROM de dados de alguns firmwares, que serão carregados no disco rígido (HD) para iniciar o sistema operacional. O sistema operacional é então carregado para a memória RAM, e parte dele é transferida para uma memória intermediária chamada de memória cache. Por fim, os dados são processados na CPU (Unidade Central de Processamento). Esse processo de inicialização é fundamental para que o computador possa funcionar e permitir que os usuários executem programas e realizem suas tarefas.
• Sim, o processo de acesso à memória RAM, à memória cache e à CPU é utilizado para todo e qualquer aplicativo ou arquivo que a pessoa abre no computador. Quando você abre um aplicativo ou arquivo, ele é carregado na memória RAM para que a CPU possa acessá-lo e processá-lo. Parte desses dados pode ser transferida para a memória cache, que atua como uma memória temporária e mais rápida, proporcionando acesso mais rápido à CPU. A memória RAM tem um papel fundamental no funcionamento do computador, pois armazena os dados necessários para a execução de programas e operações. A CPU, por sua vez, busca os dados na memória RAM para realizar as operações solicitadas pelo aplicativo ou arquivo que está sendo executado. A memória cache atua como uma memória intermediária que armazena cópias de dados frequentemente acessados pela CPU. Isso permite que a CPU acesse rapidamente esses dados, reduzindo o tempo de espera para a busca na memória RAM, que é relativamente mais lenta. Portanto, o processo de acesso à memória RAM, memória cache e CPU é uma etapa essencial na execução de qualquer aplicativo ou arquivo no computador, garantindo que os dados sejam prontamente disponibilizados para a CPU processar e fornecer uma experiência de uso mais ágil e eficiente.

Unidades de Medidas Computacionais

Bits e Bytes:
• Bits e Bytes são unidades de medida de informação digital. Um bit é a menor unidade de informação que um computador consegue manipular. Ele representa um dígito binário, podendo ser 0 ou 1, e é a base da representação de dados digitais. Por sua vez, um byte é formado por 8 bits. Ele representa uma quantidade maior de informação do que um único bit e é comumente utilizado para medir o tamanho de arquivos, armazenamento de dados e capacidade de memória em computadores. É importante compreender essas unidades, pois elas são fundamentais para descrever a quantidade de dados processados, transmitidos e armazenados em sistemas computacionais.
• Cada 0 ou 1 é chamado de bit (dígito binário).
• 1 Byte é o conjunto de 8 bits
– 01 00 10 01
– 1 letra, 1 número, 1 símbolo, 1 pontuação, 1 espaço.

Como o computador entende os comandos :
Quando você dá um comando para o computador, ele precisa ser convertido em linguagem de máquina para que o computador possa entendê-lo e executá-lo. Aqui está um resumo do processo:

  • Interpretação de comando: O sistema operacional (SO) interpreta o comando que você digitou e identifica a ação que precisa ser realizada.
  • Tradução para código de máquina: O comando é traduzido para uma série de instruções em linguagem de máquina, que o computador pode compreender e executar.
  • Execução do comando: O processador (CPU) executa as instruções em linguagem de máquina, realizando a ação requerida.
  • Resultado: O resultado da ação é exibido na tela ou armazenado em um dispositivo de armazenamento, como um disco rígido.

Em geral, esse processo acontece muito rapidamente, permitindo que o computador responda a seus comandos imediatamente. Além disso, a maioria dos sistemas operacionais inclui uma camada de abstração que esconde a complexidade da linguagem de máquina e permite que você dê comandos de uma forma mais intuitiva e fácil de usar.
